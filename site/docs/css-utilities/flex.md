## Table of Contents

## flexCenter 

Layout with flex and center horizontally and vertically.



### Output
 ```css
display: flex;
justify-content: center;
align-items: center;
```



### Example 
```typescript
import { flexCenter } from '@niftycss/css';

css({
    ...flexCenter
});
```

## flexColumn
